Как добавить радиостанции

World Machine Player умеет воспроизводить интернет-радио. Добавь станцию по имени и URL потока — ниже пошаговая инструкция для плеера и для поиска ссылок на radio-browser.info.

World Machine Player

Добавление станции в плеере

  1. Открой вкладку Radio в плеере.
  2. Нажми кнопку [+ADD RADIO].
  3. В диалоге заполни поля Name (название) и URL (ссылка на поток).
  4. Подтверди добавление — станция появится в списке, её можно выбрать и слушать.

radio-browser.info

Где взять URL радиостанции

radio-browser.info — открытый каталог интернет-радио. Найди станцию с потоком в формате MP3 и скопируй прямую ссылку.

Важно: только MP3

World Machine Player поддерживает радиопотоки в формате MP3. При выборе станции ищи тег MP3 в списке — потоки AAC, OGG и других форматов могут не воспроизводиться.

  1. Открой radio-browser.info.
  2. Перейди в раздел By category → By tag или воспользуйся поиском.
  3. Найди интересующий тег, например lofi, и открой список станций.
  4. Выбери станцию с тегом MP3 в строке списка.
  5. Нажми кнопку Save в строке станции.
  6. Браузер откроет страницу потока — скопируй URL из адресной строки (например, https://usa9.fastcast4u.com/proxy/jamz?mp=/1).
  7. Вставь скопированный URL в поле URL при добавлении радио в World Machine Player.

Если станция не играет, проверь формат (MP3) и что ссылка скопирована полностью из адресной строки после нажатия Save.
