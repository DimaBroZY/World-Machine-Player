Возможности

Плеер для своей папки с музыкой

Воспроизведение

Поддержка .ogg, .mp3, .flac и .opus, переключение треков вперед/назад, play, pause, stop и restart.

Аудио контроль

Точная громкость, изменение скорости от 50% до 150% и автоматическое зацикливание трека.

Кастомизация

Несколько визуальных тем для внешнего вида плеера и настроения.

Настройки

Можно указать свою папку с музыкой. Настройки сохраняются автоматически.

Быстрый старт

Запуск в четыре шага

  1. Скачай последнюю версию из GitHub Releases.
  2. Запусти WorldMachinePlayer.exe.
  3. Через настройки укажи папку с музыкой.
  4. Положи туда аудиофайлы, плеер просканирует их сам.
