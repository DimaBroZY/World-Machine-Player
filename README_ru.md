## ✨ О проекте

World Machine Player — фанатская реализация приложения Jukebox из OneShot: World Machine Edition.

## 🌟 Возможности

### 🎶 **Воспроизведение музыки**

* **Поддержка нескольких форматов**: воспроизводит файлы `.ogg`, `.mp3`, `.flac` и `.opus`
* **Навигация по трекам**: переключение на следующий/предыдущий трек
* **Управление воспроизведением**: воспроизведение, пауза, стоп и перезапуск

### 🎚️ **Аудио настройки**

* **Регулировка громкости**: точная настройка уровня звука
* **Регулировка скорости**: изменение скорости воспроизведения от 50% до 150%
* **Повтор трека**: автоматическое зацикливание

### 🎨 **Кастомизация**

* **Темы**: несколько визуальных тем для настройки внешнего вида плеера под настроение

### ⚙️ **Настройки**

* **Папка с музыкой**: возможность указать свою директорию с музыкой
* **Сохранение настроек**: все параметры сохраняются автоматически

## 📥 Установка

### Требования

* **Godot 4.6** или новее (для разработки)
* **Windows**

### Быстрый старт

1. **Скачайте** последнюю версию из раздела релизов
2. **Запустите** `WorldMachinePlayer.exe`
3. **Добавьте музыку**: положите файлы `.ogg`, `.mp3`, `.flac` или `.opus` в папку с музыкой
4. **Наслаждайтесь!** 🎵

### Добавление музыки

1. Нажмите кнопку **Настройки** (иконка OneShot в левом углу)
2. Укажите путь к вашей папке с музыкой
3. Добавьте аудиофайлы в эту папку
4. Приложение автоматически просканирует и загрузит музыку!
